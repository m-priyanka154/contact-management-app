To Clone and run this project 
1. Check versions:

node -v
npm -v
git --version


2. Clone the Repository
git clone https://github.com/m-priyanka154/contact-management-app.git
cd contact-management-app

3. Backend Setup
cd backend
npm install

4. Create .env file

5. Start Backend Server
npm run dev

6. Frontend Setup (New Terminal)
Open a new terminal tab/window:
cd contact-management-app/frontend
npm install
npm start
